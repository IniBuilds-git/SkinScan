About SkinScan AI

Project Overview

SkinScan AI is an advanced skin disease detection system developed as part of the CSYM015 Intelligent Systems module at the University of Northampton. This project aims to demonstrate the application of artificial intelligence in healthcare, specifically in dermatological diagnostics.

By leveraging deep learning and computer vision techniques, SkinScan AI can analyze skin images and identify potential skin conditions, providing users with educational information about their possible skin concerns.

Technical Implementation

The system is built using ResNet-50 convolutional neural network architecture, trained on a comprehensive dataset of skin disease images. The model has been fine-tuned to recognize 22 different skin conditions with high accuracy.

AI Model

  • ResNet-50 architecture with PyTorch
  • Transfer learning with ImageNet weights
  • Fine-tuned on skin disease dataset
  • Advanced data augmentation techniques
  • Weighted sampling for class imbalance
  • Over 85% validation accuracy

Technologies Used

  • PyTorch and torchvision for model development
  • Flask web framework for application deployment
  • Python for backend processing
  • HTML, CSS, and JavaScript for frontend
  • Bootstrap for responsive design
  • PIL (Pillow) for image processing

Dataset & Skin Conditions

The model was trained on a comprehensive dataset of dermatological images containing 22 different skin conditions. The dataset includes over 13,000 training images with balanced representation across all classes.

Common Conditions
  • Acne
  • Eczema
  • Psoriasis
  • Rosacea
  • Vitiligo
  • Tinea (Ringworm)
Serious Conditions
  • Skin Cancer
  • Melanoma
  • Actinic Keratosis
  • Basal Cell Carcinoma
  • Lupus
  • Vasculitis
Other Conditions
  • Benign Tumors
  • Vascular Tumors
  • Seborrheic Keratoses
  • Warts
  • Moles
  • Candidiasis

Note: The dataset was sourced from Kaggle's "Skin Disease Dataset" which contains images collected from various medical repositories with proper permissions and ethical considerations.

Ethical Considerations & Safety

Important Medical Disclaimer

This system is designed for educational and research purposes only. It should never be used as a substitute for professional medical diagnosis or treatment. Always consult qualified healthcare professionals for medical advice.

Ethical Implementation
  • Privacy Protection: Uploaded images are processed locally and automatically deleted after analysis
  • Bias Mitigation: Training dataset includes diverse skin tones and demographic representation
  • Transparency: Confidence scores are provided to indicate prediction reliability
  • Accessibility: User-friendly interface designed for various user groups
  • Educational Focus: Clear messaging about the tool's limitations and proper use
